# bouncing-ball

## Overview
This is a simple bouncing ball simulation built in pygame.

## Preview
![bouncing-ball.gif](bouncing-ball.gif)

## Contents
This section goes over the contents in this directory.
- ```venv```: virtual environment
- ```requirements.txt```: package list
- ```circle.py```: broken file, ignore
- ```main.py```: contains code for the bouncing ball simulation

## Setup
In this section, we are assuming that the user has some knowledge of Python and has setup a working environment.

1. Install the required Python packages inside the virtual environment (venv).

2. Run ```main.py``` through commands or the run button in your IDE.

## Notes
- This program runs at 60 fps, however the gif shown above is recorded at 24 fps (Windows Snippet Tool).
