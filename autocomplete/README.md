# autocomplete

## Overview
This is a autocomplete tool built in pygame.

## Preview
![autocomplete.gif](autocomplete.gif)

## Contents
This section goes over the contents in this directory.
- ```requirements.txt```: package list
- ```main.py```: contains code for the autocomplete tool
- ```utility.py```: contains code to load words list and find matchings

## Setup
In this section, we are assuming that the user has some knowledge of Python and has setup a working environment.

1. Install the required Python packages inside the virtual environment (venv).

2. Download or create a ```words.txt``` file containing a word in each line.

3. Run ```main.py``` through commands or the run button in your IDE.

## Notes
- This program runs at 60 fps, however the gif shown above is recorded at 24 fps (Windows Snippet Tool).
- The virtual environment is not included in this repository.
- The list of words: ```words.txt``` is not included in this repository, but can be downloaded below under the sources section or through another source.
