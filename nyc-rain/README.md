# nyc-rain

## Overview
Is it raining in NYC right now?

## Description
View the status at ```rain.csv```.

## Tools
- [Open-Meteo API](https://open-meteo.com/)

## Notes
Updates are made every hour in GitHub while Open-Meteo takes around 15 minutes.
This API is free, requires no sign up, and provides a starter template to use.

## Comment
This is a starter template to work with GitHub Actions CI/CD.
